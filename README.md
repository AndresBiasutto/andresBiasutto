# 👋 ¡Hola! Soy Andrés Biasutto

**Ingeniero de Software / Desarrollador Full-Stack**

Estudiante avanzado de **Análisis de Sistemas** con fuerte enfoque en el desarrollo de aplicaciones web modernas, escalables y de alta calidad. Me apasiona construir sistemas completos con buena arquitectura, código limpio y excelentes prácticas de desarrollo.

---

### 🛠️ Tecnologías y Stack

**Backend**
- Node.js + Express
- TypeScript (project-wide con strict mode)
- PostgreSQL + Sequelize ORM
- Autenticación JWT + bcrypt
- Arquitectura en capas (routes → middlewares → controllers → services → repositories → models)

**Frontend**
- React + Vite + TypeScript
- Redux Toolkit
- Tailwind CSS

**Calidad y DevOps**
- Testing: Jest + Supertest
- Validación estática y scripts de validación (`npm run validate`)
- CI/CD con GitHub Actions
- Middlewares reutilizables (validación, ownership/authorization)
- Manejo centralizado de errores

---

### 🚀 Proyecto Destacado

**Campus Virtual** — Sistema de gestión educativa full-stack (monorepo)

Desarrollé una plataforma completa para la administración de cursos, alumnos y docentes con:

- Backend robusto en TypeScript con arquitectura limpia y escalable
- Frontend moderno con React + Redux Toolkit
- Autenticación segura, validaciones estrictas y middlewares de ownership
- Testing automatizado y pipeline de CI

Este proyecto refleja mi capacidad para diseñar y mantener aplicaciones profesionales con foco en mantenibilidad y calidad de código.
